Preview of the Eawag Research Data Institutional Repository (ERIC)

This is a preview of the the new release of Eawag's research data repository. It features a new meta-data schema that is better adapted to user's needs, improved workflow support, new functionality such as restricted resources, usability improvements and bug fixes.

Usage

The preview phase is for real-world testing. Please

  • Explore the system to your heart's content
  • Upload dummy packages, try out all the features you can find.
  • Try weird things you normally wouldn't.
  • Please send bug reports and usabilty problem reports to rdm@eawag.ch.

NO REAL DATA

After the preview phase, the database will be cleaned. Do not put data exclusively here.

For real packages, use the service at https:eaw-ckan-dev1 .

Underlying software

  • The CKAN has been upgraded from version 2.5.2 to version 2.7.3 .
  • The search platform SOLR has been upgraded from version 5.3.1 to version 6.5.0 .
  • The database Postgres has been upgraded from version 9.4.15 to version 9.6.9 .

Server

Instead of on a single virtual machine (VM), Postgres and SOLR now run on a separate server. The storage backend is now connected to the new storage system at Eawag which provides a total storage capacity of 11 TB for ERIC.

Timeline (anticipated)

  • End of preview: 2018-08-19
  • Final bug-fixes and polishing: 2018-08-20 to 2018-08-26
  • Start of operational phase (you will be notified): 2018-08-27
